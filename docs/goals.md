# Goals

## Needs:
1. Create a calculator for converting mm to inches and inches to mm.
2. Have options for mathematical operations (+,-,*,/)
3. Have ability to enter values with a given unit (mm or in).
4. Raw PCB board design, with a focus on using only SMD components.
5. OLED display.
6. Small form-factor design (~2x3in).
7. USB-C or Battery power.
8. 4x4 button matrix
9. On/Off switch

## Wants:
- Black PCB
- Sleep function, auto wake on button press
